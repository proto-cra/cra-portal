CRA account help - Businesses

GST/HST account overview – CRA account help - Businesses

You can complete many GST/HST (or some QST) related task in the your CRA account. In the main 3 tabs in the center of the overview page, you will be able to complete tasks related to return, rebates and reporting and tax options (called elections). Several other actions such as closing your GST account or request interest relief can also be found in the right services menu or in program accounts.

On this page

  • Returns
  • Rebates
  • Reporting and tax options (elections)

Returns

Use the Returns tab to:

  • See upcoming expected and next returns (including due dates)
  • Review filed returns (assessed, reassessed, adjusted, etc.)
  • Start common actions such as filing a new return, viewing details, or adjusting a return

Common actions

  1. File a return
    • Select File a return.
    • Follow the steps to complete and submit the return for the appropriate reporting period.
  2. View return details
    • Select the return you want (usually by clicking the Reporting period link in the filed returns table).
    • Select View return details to see more information.
  3. Adjust a return
    • Select the return you want to change (as above).
    • Select Adjust a return and follow the steps to submit an adjustment.

Rebates

Purpose

Use the Rebates tab to:

  • Check the status of rebate submissions.
  • Open details for certain rebate records.
  • Start a new rebate request or adjust specific rebate types (when available).

Rebate status

The Rebate status table lists rebate submissions and their current status.

The page indicates: click Period covered to view additional details (only available for rebate type GST66/GST284).

The tab also notes that detailed information on your rebate claim may be found on your Notice of Assessment.

Common actions

  1. Open a rebate record (when available)
    • Look for a rebate where the Period covered value is clickable (eligible rebate types only).
    • Select the linked period to view additional details.
  2. File a rebate
    • Select File a rebate.
    • Complete the rebate request and submit.
  3. Adjust a PSB rebate
    • Select Adjust a PSB rebate.
    • Follow the steps to submit the adjustment.

Reporting and tax options (elections)

Purpose

Use the Elections tab to:

  • Review your current GST/HST account settings that are controlled by elections (for example, reporting period and fiscal year end).
  • File a new election when you need to change one of the listed options.
  • View previously filed elections.

Election summary

The Elections tab includes an Election summary area. Examples of the items shown include:

Reporting period (GST20)
Your current reporting frequency (for example, quarterly).
Fiscal year end (GST20)
Your current fiscal year end date (shown as a month and day).
Quick method of accounting (GST74)
Status of the quick method election (for example, an election may be active or revoked with an effective date).
NIL consideration (RC4616)
A link may be provided to view elections related to treating certain taxable supplies as nil consideration for GST/HST purposes.

Note: The tab states that to change any of the listed options, you must file an election.

Common actions

  1. File an election
    • Select File an election.
    • Choose the election type you need and follow the prompts to submit.
  2. View your elections
    • Select View your elections to see elections currently on file and any available details.
  3. View elections for NIL consideration
    • In the NIL consideration (RC4616) row, select the available link (for example, View elections) to open details.
